示例使用:



    创建一个两节点节点集群，其中一个master节点（用来控制集群），一个node节点（用来运行工作负载，比如 Pods 和 Deployments)。

    ┌──────────────────────────────────────────────────────────┐
    │ 在第一个节点上:                                           │
    ├──────────────────────────────────────────────────────────┤
    │ master# kubeadm init                                     │
    └──────────────────────────────────────────────────────────┘

    ┌──────────────────────────────────────────────────────────┐
    │ 在第二个节点上:                                           │
    ├──────────────────────────────────────────────────────────┤
    │ node# kubeadm join <init 返回的参数>                      │
    └──────────────────────────────────────────────────────────┘


    然后你可以在任意多个其它机器上重复第二个步骤。
